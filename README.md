# HotelManagementSystem
JavaFX ve MySQL kullanılarak geliştirilen kapsamlı Otel Rezervasyon ve Yönetim Sistemi. Yazılım Mimarisi dersi kapsamında geliştirilen bu proje; Singleton, Factory, Observer, Decorator, State ve Strategy tasarım desenlerini, CRUD işlemlerini ve rol tabanlı (Admin/Müşteri) yetkilendirme mimarisini içerir.
